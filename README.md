# Zapret Manager - `софт для обновления Zapret и TgWsProxy от FlowSeal в России автоматически. (По нажатию 1 кнопки или по нажатию одного тумблера)`

##

**_Самое важное: скачивать с этим приложением zapret и TgWsProxy лучше заново через само приложение - как это сделать указано ниже_**

##

## Главное окно приложения

![](https://i128.fastpic.org/big/2026/0715/83/bec64c1be7aa05a88628f3b77cca4283.png)

**Здесь указаны: актуальность версии и сама версия, также тут можно сменить и мод приложения.**
**Далее будет объясняться логика работы приложения, но здесь существует также кнопка `Update` _(она нужна для того, чтобы свапнуться между версиями вручную)_**

## Окно настроек

![](https://i128.fastpic.org/big/2026/0715/12/1f82f413e12ec02427ead60d61cc8f12.png)

**Вся работа с приложением находится здесь: вы можете ставить автоапдейт, автоматически открывать нужное приложение или же при обновлении `Zapret` сохранять весь `list-general.txt` (домены), чтобы не переносить их вручную. И всё автоматически или по одной кнопке.**

### Выбор директории

Чтобы выбрать нужную папку под приложение, создайте в ней файл `new.txt`
