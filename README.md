# Zapret Manager - `утилита для обновления Zapret и TgWsProxy от FlowSeal в России автоматически. (По нажатию 1 кнопки или по нажатию одного тумблера)`

**Перетаскивание окон приложения осуществляется по удерживанию на тексте в хедере (сверху) - `Any Mode` или `Settings`.**
**_Самое важное: скачивать с этим приложением `Zapret` и `TgWsProxy` лучше заново через само приложение - как это сделать указано ниже ||_ __Известные проблемы у приложения также указаны внизу!__ _Также приложение при первом запуске, поставит себя в автозагрузку, для автозапуска `Zapret/TgWsProxy` по вашему выбору._**

## Окно настроек

![](https://i128.fastpic.org/big/2026/0715/12/1f82f413e12ec02427ead60d61cc8f12.png)

**Вся работа с приложением находится здесь: вы можете ставить автоапдейт, автоматически открывать нужное приложение или же при обновлении `Zapret` сохранять весь `list-general.txt` (домены), чтобы не переносить их вручную. И всё автоматически или по одной кнопке.**

### Выбор директории

**Чтобы выбрать нужную папку под приложение, создайте в ней файл `new.txt` и выберите его. | Чтобы выбрать именно директорию текущего `Zapret/TgWsProxy`: смените фильтр и выберите именно нужный файл, который планируете, чтобы в будущем автоматически запускался.**

![](https://i128.fastpic.org/big/2026/0715/89/c760c5a0dd7853251b24a51094b3d089.png)

### Выбор версий

![](https://i128.fastpic.org/big/2026/0715/d1/e5a48dd10f45faa5a2b29c35a91ff7d1.png)

**После выбора директорий: нужно указать нужную версию `Zapret/TgWsProxy` (самая актуальная всегда сверху). ! Здесь заключается одна из проблем: если не перекачивать `Zapret/TgWsProxy` через приложение, то версию будет определить невозможно и приложение укажет о том, что ваша версия неактуальна, даже если актуальна и при установке галочек AutoUpdate, приложение всё равно их обновит, будьте внимательны !**
